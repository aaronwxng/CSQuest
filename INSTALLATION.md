# Installation Guide

## Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Servers**
   ```bash
   npm run dev
   ```
   
   This will start both:
   - Client on `http://localhost:3000`
   - Server on `http://localhost:3001`
